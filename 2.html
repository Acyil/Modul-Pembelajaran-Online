Praktikum: Contoh 1 — ON/OFF Solenoid

Durasi: 30–45 menit

Alat & Bahan

  • Trainer PLC
  • Kabel jumper
  • Solenoid valve (1)
  • Push button dan sensor

Langkah-langkah

  1. Matikan power. Sambungkan input tombol ke I0 dan output solenoid ke Q0.
  2. Nyalakan power. Buka software PLC dan hubungkan ke trainer.
  3. Tulis program: jika tombol ditekan maka Q0 = 1.
  4. Download program ke PLC, jalankan mode RUN.
  5. Amati aksi solenoid. Catat waktu respons dan masalah jika ada.

Isi Lembar Kerja: waktu ON/OFF, kesalahan wiring, dan refleksi perbaikan.
